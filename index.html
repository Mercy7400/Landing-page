Key Features

Feature 1: Shorten URLs

Easily shorten long URLs to a more manageable size with a click of a button.

Feature 2: Custom Aliases

Create custom short links that are easy to remember.

Feature 3: Analytics

Track how many times your shortened URLs have been clicked.

About the Project

This project was inspired by the need to manage long URLs, simplify sharing, and track analytics. It was built as part of a Portfolio Project for Holberton School.

Timeline: The project began in August 2024 and was completed in September 2024.
